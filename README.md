# TR6 Restomod Platform

  A full-stack web application for TR6 restomod enthusiasts to discover, compare, and track aftermarket parts across multiple suppliers.

  ## Features
  - Parts catalog with multi-supplier aggregation
  - Community build threads
  - User marketplace for buying/selling parts & cars
  - Multi-currency pricing support
  - Wishlist and garage management

  ## Tech Stack
  - React 18 + TypeScript + Vite
  - Express.js backend
  - PostgreSQL + Drizzle ORM
  - Tailwind CSS + shadcn/ui
  - Replit Auth (OpenID Connect)
